# 1. 텍스트 분류

- 텍스트를 입력받아, 텍스트가 어떤 종류의 범주(Class)에 속하는지를 구분하는 작업

- ex) 스팸 메일 분류

  > ![image-20200114180219254](images/image-20200114180219254.png)



### 텍스트 분류 유형

- 이진 분류 : 범주가 2가지
- 다중 클래스 분류 : 분류해야 할 범주가 3가지 이상

<br>

<br>

### 다양한 텍스트 분류

- 감성 분석 : 리뷰가 긍정리뷰인지 부정 리뷰인지 분류
- 의도 분석 : 사용자의 의도를 질문, 명령, 거절과 같은 클래스로 분류
